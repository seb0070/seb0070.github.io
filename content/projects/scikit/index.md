리듬에 맞춰 노트를 입력하는 웹 기반 리듬 게임 프로젝트입니다.  
JavaScript와 HTML Canvas를 활용하여 **실시간 키 입력 처리**, **애니메이션 렌더링**,  
**노트 타이밍 판정 알고리즘** 등을 구현했습니다.  
단순히 기능 구현을 넘어서,  
사용자가 **리듬감과 시각적 피드백을 동시에 느낄 수 있는 UX**에 중점을 두었습니다.
